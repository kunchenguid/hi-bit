Lesson resume helper covered by doctrine

Checkpoint every answer

LessonProgress.checkpoint("0001", { round, score }) saves mid-lesson state after every meaningful step, not only at completion.

Resume after restart

LessonProgress.resume("0001", { round: 1, score: 0 }) restores the same question and score when the page opens again.

Finish clears checkpoint

LessonProgress.finish("0001", { bestScore }) marks the lesson done for the hub and starts a replay fresh.
